# Modern Calculator

A modern, responsive calculator built with HTML, CSS, and JavaScript. Features:

- Attractive glassy UI with light/dark themes
- Basic arithmetic, parentheses, percent, and exponentiation
- Math functions: sin, cos, tan, sqrt, ln, log (base 10)
- DEG/RAD toggle for trigonometry
- Live preview and safe-ish evaluation
- History panel with persistence
- Memory stack with multiple slots (push/pop/add/subtract/clear)

Usage

1. Open `index.html` in a browser (or serve with a local static server).
2. Use the buttons or keyboard to build expressions.
3. Press Enter or `=` to evaluate and save to history.
4. Use MPush to push the current value to memory, MPop to pop the top slot, M+/M- to add/subtract to the top slot, and MC to clear all memory.

Files

- index.html — HTML markup and UI
- styles.css — styles and themes
- script.js — application logic
